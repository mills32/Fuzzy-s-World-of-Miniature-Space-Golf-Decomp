Fuzzy's World of Miniature Space Golf Decomp (in progress)
----------------------------------------------------------

At the moment, only some utils were coded:
  - player.c: Plays original loudness tracker and converts patterns, notes and most instrument parameters to impulse tracker (converted .it files will be added).
  - ppext.c: Pixel painters extractor, extract files from .RES resources.
  - convert.c: Converts.SPF and .ANI to .GIF.
